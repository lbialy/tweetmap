Twitter stream map

Hi, this project is a small show of capabilities of Vert.x reactive platform. Usage: enter keywords that you want to track into the input box below the map. Good choice of keywords for geolocation of tweets is: "work java scala" due to many tweets actually having geotags.

Also, twitter streaming api is rate limited. If keywords are changed too often (what causes stream reconnection due to filter change) app may hit filter request rate limit. If such thing happens stream will be automatically reconnected after rate limit expires.

Libraries and frameworks used: Vert.x, Immutables, Rx.java, Javaslang on the backend and Angular.js and Rx.js on the frontend.

Source code is available at: GitHub.
